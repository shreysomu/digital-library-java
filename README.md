# 📚 Digital Library System (Core Java)

## 🚀 Features
- Add Book
- View Books
- Search Book by ID
- Update Book
- Delete Book
- Menu-driven console application

## 🛠 Tech Stack
- Java
- OOP Concepts
- Collections (ArrayList)
- Exception Handling

## 📌 Description
This is a console-based Digital Library application built using Core Java.
It allows users to manage books with CRUD operations in a structured way.

## ▶️ How to Run
1. Clone the repo
2. Run Main.java
3. Follow menu options
